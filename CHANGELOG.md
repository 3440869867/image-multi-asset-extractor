# Changelog

## 0.1.0

- Added standalone repository structure.
- Added CLI extraction pipeline for PNG/JPG/WEBP sprite sheets.
- Added NVIDIA CUDA environment check and explicit AMD GPU unsupported notice.
- Added optional Gradio Web UI.
- Added CPU dependency profile for Windows, Linux, macOS, and Docker.
- Added best-effort AMD profiles for Windows DirectML and Linux ROCm.
- Added backend selection through `gpuBackend`.
- Added cross-platform install/run scripts.
- Added CPU-first Dockerfile and docker-compose setup.
- Added generated manifest output for map asset imports.
